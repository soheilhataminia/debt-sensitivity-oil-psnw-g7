# Public Debt Sensitivity to Oil-Price Conditions in the G7

This repository contains the replication data and EViews code for the manuscript **“Public Debt Sensitivity to Oil-Price Conditions in the G7: The Role of Public Sector Net Worth.”** The study uses a balanced annual G7 panel for 2000–2020 to examine whether public sector net worth moderates the sensitivity of general government gross debt to common oil-price conditions.

## Files

- `data/raw/`: raw source files downloaded from official providers.
- `data/processed/data_processed_g7.xlsx`: final processed panel dataset.
- `code/eviews/reproduce_pipeline_eviews.prg`: EViews replication program.
- `data_dictionary.md`: variable definitions and data notes.
- `DATA_LICENSE.md`: data-use and source-license information.
- `CITATION.cff`: citation information.

## Sources

The data come from IMF WEO, IMF PSBS, IMF DataMapper, BIS, EIA, and World Bank WDI. Raw data remain subject to the terms and conditions of the original providers.
