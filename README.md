# Public Debt Sensitivity to Oil-Price Conditions in the G7

Replication data and EViews code for the manuscript:

**Public Debt Sensitivity to Oil-Price Conditions in the G7: The Role of Public Sector Net Worth**

## Overview

This repository provides the data and EViews replication code for a study of whether prior public-sector balance-sheet strength moderates the sensitivity of public debt to common oil-price conditions in G7 economies.

The analysis uses a balanced annual panel of G7 countries over 2000–2020. The dependent variable is general government gross debt as a percentage of GDP. Public sector net worth is used as the main indicator of public-sector balance-sheet strength. Brent oil prices are interacted with lagged public sector net worth in a dynamic two-way fixed-effects panel model.
