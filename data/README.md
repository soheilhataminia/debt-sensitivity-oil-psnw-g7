# Data

This folder contains the data files used in the replication package.

## Folder structure

- `raw/`: original source files downloaded from official data providers.
- `processed/`: final cleaned panel dataset used in the EViews replication program.

## Processed dataset

The main processed dataset is:

`processed/data_processed_g7.xlsx`

It contains the balanced annual G7 panel used in the manuscript for 2000–2020.

## Raw data sources

The raw data files include source data from the IMF, BIS, EIA, World Bank WDI, and IMF PSBS database. These files are retained in their original downloaded formats for transparency and replication.

Raw data remain subject to the terms and conditions of the original data providers.
